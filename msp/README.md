### `msp`模块：飞控的通信协议

`MSP`（MultiWii Serial Protocol）是飞控与地面站、外设交互的标准协议，核心作用是实现 “飞控参数读写、状态数据传输”：

| 文件                 | 作用                                                         |
| -------------------- | ------------------------------------------------------------ |
| `msp_protocol*.h`    | MSP 协议定义：包含 Betaflight 扩展的 MSP V2 协议格式，统一飞控与外部设备的通信规则。 |
| `msp_serial.c/h`     | MSP 串口实现：通过串口（UART/USB）传输 MSP 数据，是地面站（如 Betaflight Configurator）调参、读取飞控状态的核心接口。 |
| `msp_build_info.c/h` | 版本信息交互：提供飞控固件版本、硬件信息的 MSP 查询接口，方便地面站识别飞控配置。 |
